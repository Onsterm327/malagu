INVALID_CLIENT

INVALID_CLIENT: = "invalid_client"
  • Defined in packages/oauth2-core/src/node/error/error-protocol.ts:79

{@code invalid_client} - Client authentication failed (e.g., unknown client, no client authentication included, or unsupported authentication method). The authorization server MAY return a HTTP 401 (Unauthorized) status code to indicate which HTTP authentication schemes are supported. If the client attempted to authenticate via the "Authorization" request header field, the authorization server MUST respond with a HTTP 401 (Unauthorized) status code and include the "WWW-Authenticate" response header field matching the authentication scheme used by the client.
